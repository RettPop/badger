# SMatcher

A Flutter-based organizing puzzle game with six game modes. Swap tiles on a 4x5 grid to match groups of 3 or more by color, letter, or badge value. Each attribute match provides a different score multiplier, and matching multiple attributes in a single group stacks the multipliers for high-scoring plays.

## Game Modes

- **Convenient** -- Standard gameplay with adjacent tile swaps and optimum score tracking.
- **High Scores** -- Constrained tile generation ensures high optimum scores (at least 50) for more rewarding matches.
- **Drop Down** -- Gravity-based tile refill: tiles fall down and shift laterally after matches are cleared.
- **Simple Drag** -- Drag any tile to any position on the board for unrestricted swaps.
- **Snake Drag** -- Drag a tile across the board in a snake path, shifting tiles along the way.
- **Arcade** -- Race against time as your score drains by 1 point every second.

## Getting Started

This project is built with Flutter. To run:

```bash
flutter run
```

To run tests:

```bash
flutter test
```
